# 🗂️ HR Leave Management System

A web-based HR workflow tool built with Python and Flet — designed to eliminate manual leave tracking and streamline approval processes for teams.

> **Built to automate a real operational bottleneck:** manual leave tracking via email/spreadsheets wastes HR time and creates approval delays. This app replaces that workflow with a structured, auditable system.

---

## 🚀 Live Demo

> Run locally in under 2 minutes (see setup below)

---

## ✨ What It Does

| Feature | Description |
|---|---|
| 🔐 Auth | Login system with employee/manager roles |
| 📝 Leave Requests | Employees submit requests with date picker & leave type |
| ✅ Approval Workflow | Managers approve or reject with one click |
| 📋 History Tracking | Full audit trail of all leave decisions |
| 🌐 Web Interface | Runs in browser via Flet — no frontend framework needed |

---

## 🛠️ Tech Stack

- **Python 3.x** — core logic
- **Flet** — Python-to-web UI framework (no JS required)
- **JSON** — lightweight data persistence

---

## ⚡ Quick Start

```bash
# 1. Install dependency
pip install flet

# 2. Run the app
python main.py

# 3. Open in browser
# → http://localhost:8550
```
